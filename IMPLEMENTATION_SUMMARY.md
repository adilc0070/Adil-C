### GitHub Statistics
- Check rate limits in console
- Private repos won't show stats
- Consider GitHub personal token for higher limits
- Cache results to reduce API calls

- Convert to WebP format
- Compress with tools like TinyPNG
- Use proper dimensions (800x600 recommended)
- Keep file sizes under 200KB
